== README

Simple application to search github repositories, requiring github auth

* Ruby version 2.3.0

* Rails version 4.2.5

* git clone https://github.com/Steinitch/gh_search_heroku.git

* cd gh_search_heroku

* bundle exec rake db:migrate

* bundle exec rake db:test:prepare

* bundle exec rspec

* rails s

* navigate webbrowser to localhost:3000
